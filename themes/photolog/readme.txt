A theme called Photolog.

== Description ==

Photolog is a Photo Blogging theme which can be used both by bloggers or Photographers. It has a simple two column design with the Featured Image on one side and the content on the other. The Image gives you a preview of what your post is about and will attract readers to read the well laid out content with enough white space. A lot of effort has been given to make your site easy to read and make the design look good on all devices. For mobile and smaller devices the layout squeezes to one column layout so the reading experience is not compromised.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Features ==
1. Custom Menus, Primary menu on the top, Social Menu at the bottom
2. 3 Footer Widgetized section
3. Custom Header Support
4. Custom Background Support
5. Post Formats
6. JetPack Custom Logo

= Recommended plugins =

To achieve the best of the theme, please install and activate JetPack plugin and some of its submodules such as Custom Logo and Infinite Scroll. You can further customize your theme by visiting Appearance -> Customize menu in  your wp-admin page and change the needed options.
